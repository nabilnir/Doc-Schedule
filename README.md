# Doc Schedule

Doc Schedule is a professional healthcare management platform built for doctors and patients. It streamlines the appointment process through intelligent automation and a premium user interface.

## Core Features

* Intelligent Slot Generation for automated clinic availability
* Google Calendar Synchronization for real time scheduling
* Secure Payment Gateway integration via Stripe
* Patient Intake Forms for digital information gathering
* Automatic Email Reminders to reduce missed appointments
* Professional PDF Reports generation
* Responsive Admin Dashboard with analytics
* Smooth Animations powered by GSAP

## Technology Stack

* Next.js 15 plus React 19
* Tailwind CSS version 4
* GSAP for advanced animations
* MongoDB and Mongoose for data persistence
* Firebase for client side services
* NextAuth for secure authentication
* Stripe for payment processing
* Google Calendar API integration
* Nodemailer and Node Cron for automated tasks

## Getting Started

Follow these steps to run the project locally.

1. Clone the repository to your local machine
2. Install dependencies using npm install
3. Configure your environment variables in a .env file
4. Start the development server using npm run dev
5. Open http://localhost:3000 in your browser

## Project Structure

* src/app: Application routes and pages
* src/components: Reusable UI components and sections
* src/lib: Database and API configurations
* src/models: Mongoose schemas and data models
* public: Static assets and brand logos
